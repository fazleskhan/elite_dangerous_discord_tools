# Pathing in Elite Dangerous GIS
#### Video Demo:
#### Description:

The video game Elite Dangerous attempts to model theMilky Way in three dimensional space. This project is to provide a command line tool to path routes between systems (i.e. nodes) in games' GIS system.

The project leverages data dumps generated by the Neutron Planner website and a query API provided by Elite Dangerous Galactic Information System (EDGIS).

https://www.spansh.co.uk/dumps

https://edgis.elitedangereuse.fr/

https://github.com/elitedangereuse/edgis

In order to be a good net citizen, data is cached locally in a Tinydb database for reuse. The project provides an initial preloaded chunck of nodes centered around the Sol system as well additional nodes queried through use.

The GIS data does not change so updates are not required. However the original dataset is very large and is not feasible to host locally. The current full dataset is 99.7GiB compressed.

The ultimate goal for this project is to reuse this logic in a discord bot thus is is implemented as a library for reuse. The elements of the project are split into different source files based on thier functional area. Each source file has a test file to excerise it's logic.
